Registry
What is Registry?
  Registry is The Central DataBase System Maintained By Windows
  For Saving Data Such Like Desktop-Resolution, WallPaper & Other
  Control-Panel Settings Like Mouse-Speed, Default Browser etc.
  That is Why You always Get Same Wallpaper which you have set
  Earlier even after a Restart.
Registry Functions
  Say You want to Save Some Settings For Your application such as
  Last View of The Report Or the Best Example is Most Recently Used Files.
  Visual-Basic Provides Such Functions For Working with Registry.
  For Saving Data SaveSetting Function is Used
  For Reading Data GetSetting Function is Used
  And For Deleting Data DeleteSetting Function is Used
  I have Used This Functions In This Project So you can See it.

Creating .Reg Files
  Many Times We See A File Which Adds Information to the Registry
  First Of All You Have To Decide, Where That Reg File Will Add
  Information. After That Open Notepad and Type The Following:-
  REGEDIT4
  Leave One Line After Writing This
  Than Write The Path Where The Information Will Be Stored
  Now on the Next Line Write the "Key" = "KeyValue"
  Here Key Will be The Key You are Referencing To
  And KeyValue Will Be the Value You Want To Add/UpDate
